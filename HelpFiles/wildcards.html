Using Wildcards in Name Filter

Ark Notes Supports the use of SQL wildcards in its search filter allowing you to search for dinos based on partial names (including Dino names, persons names and tribe names). there are 2 types of wildcards you can use to represent part of your search.

  • % Can represent 0 or many characters in the filter  for example Al% could return Alex, Al Or Allen 
  • _ Can represent 1 character in the filter for example Stev_ could return Steve but not Stevie
